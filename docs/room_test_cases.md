# WPF 강의실 배정 정책 및 테스트 케이스

이 문서는 `wpf/` 현재 구현의 강의실 배정 정책을 정리한다. 시간표 생성은 분반 단위로 이루어지며, 강의실은 과목 설정을 우선한다.

## 핵심 정책

| 과목 상태 | 강의실 처리 | 적용 제약 |
|---|---|---|
| `FixedRooms`가 1개 이상 | 지정된 모든 방을 해당 수업 시간에 동시에 점유한다. 교수 강의실 조건보다 과목 설정이 우선한다. | HC-14 |
| `FixedRooms`가 비어 있음 | 방 1개를 자동 배정한다. 담당 교수의 `AllowedRooms`와 `UnavailableRooms`를 후보 조건으로 적용한다. | HC-21 |
| 같은 과목의 분반이 2개 이상이고 모두 자동 배정 | 모든 분반에 하나의 공통 강의실을 선택한다. | HC-22 |
| 같은 과목 분반 중 하나라도 `FixedRooms`가 있음 | HC-22를 적용하지 않고 각 분반의 과목 고정 강의실 규칙을 따른다. | HC-14 |

- HC-21은 **교수의 여러 과목을 한 강의실로 묶지 않는다.** 교수의 강의실 조건은 자동 배정 과목의 후보를 제한하는 용도다.
- `FixedRooms`에 여러 방이 있으면 선택 후보가 아니라 동시 점유 방 목록이다.
- `IsFixed`는 시간만 고정한다. 강의실은 `FixedRooms`가 있으면 HC-14, 없으면 HC-21/HC-22로 결정한다.

## Cross와 강의실

Cross는 대응 분반을 같은 시간에 배치한다. 따라서 대응되는 두 분반이 공통 고정 강의실을 사용하면 같은 시간에 같은 방을 점유하게 되어 불가능하다.

| 상황 | 처리 |
|---|---|
| Cross 추가 시 대응 분반의 `FixedRooms`가 겹침 | 저장하지 않고 `IE-039`를 표시한다. |
| 기존 데이터에 같은 조건의 Cross가 존재 | 해 생성 전 `GE-028`을 표시한다. |
| 두 과목이 자동 배정이고 공통 후보 방만 있음 | 차단하지 않는다. 솔버가 서로 다른 방을 배정할 수 있으면 해를 만든다. |

## 테스트 케이스

### TC-01. 같은 교수의 자동 과목은 서로 다른 방을 사용할 수 있음

**입력**: 같은 담당 교수의 과목 A, B가 모두 `FixedRooms=[]`이고 강의실이 2개 이상이다.

**기대**: A와 B는 교수의 허용/불가 강의실 조건만 만족하면 되며, 서로 다른 방을 사용해도 정상이다.

**확인**: 같은 교수라는 이유만으로 방을 하나로 강제하지 않는다.

### TC-02. 자동 배정은 교수 강의실 조건을 따른다

**입력**: 자동 과목의 담당 교수 `AllowedRooms=[R1,R2]`, `UnavailableRooms=[R2]`.

**기대**: 해당 과목은 R1만 사용한다.

**확인**: `AllowedRooms` 밖 또는 `UnavailableRooms` 안의 방을 사용하지 않는다.

### TC-03. 과목 고정 강의실이 교수 조건보다 우선한다

**입력**: 과목 `FixedRooms=[R1]`, 담당 교수 `AllowedRooms=[R2]` 또는 `UnavailableRooms=[R1]`.

**기대**: 과목은 R1을 사용한다.

**확인**: 과목에 방이 명시된 경우 교수 강의실 조건 때문에 해가 불가능해지지 않는다.

### TC-04. 같은 과목의 자동 분반은 공통 강의실을 쓴다

**입력**: A-01, A-02가 모두 `FixedRooms=[]`이고 배정 가능한 방이 2개 이상이다.

**기대**: A-01과 A-02의 모든 수업은 같은 하나의 강의실을 사용한다.

**확인**: 한 분반의 시간대마다 방이 바뀌거나, A-01과 A-02가 서로 다른 방을 쓰면 실패다.

### TC-05. 다중 고정 강의실은 동시 점유한다

**입력**: 과목 `FixedRooms=[R1,R2]`, 주당 2시간.

**기대**: 수업이 있는 각 시간에 R1과 R2를 모두 점유한다.

**확인**: 총 점유 수는 `주당 시수 × 2`이고, 한 방만 사용하는 배정은 실패다.

### TC-06. 공통 고정 강의실 Cross는 즉시 차단한다

**입력**: A-01/A-02와 B-01/B-02가 모두 R1을 고정 강의실로 사용한다.

**기대**: Cross 추가 시 `IE-039`가 표시되고 Cross는 저장되지 않는다.

**확인**: 기존 데이터에 동일 조건 Cross가 있으면 생성 전 `GE-028`이 표시된다.

### TC-07. 자동 배정 Cross는 방을 분리해 배정할 수 있다

**입력**: A, B가 Cross이고 각 과목의 분반은 모두 자동 배정이며, 동시에 사용 가능한 방이 2개 이상이다.

**기대**: 대응 분반은 같은 시간에 배치되고, 그 시간에는 서로 다른 강의실을 사용한다. 각 과목 내부 분반은 HC-22에 따라 공통 강의실을 유지한다.
