# 수동편집 위반사항 표시 정책

이 문서는 수동편집 화면에서 같은 위반사항이 어디에 표시되는지 정리한 표입니다.

- 시간표 작은 글자: 블록 오른쪽 아래의 빨간 작은 글자
- 오른쪽 위반사항: 수동편집 화면 오른쪽의 `제약조건 위반사항`
- 전체 검증: `전체 검증` 버튼을 눌렀을 때 나오는 검사 항목

## 현재 표시 기준

시간표 작은 글자에 표시되는 항목을 기준으로 오른쪽 위반사항과 전체 검증을 맞춘다.

| 위반 항목 | 시간표 작은 글자 | 오른쪽 위반사항 | 전체 검증 |
|---|---|---|---|
| 강의실 중복 | 강의실 중복 | 표시 | 표시 |
| 교수 중복 | 교수 중복 | 표시 | 표시 |
| 분반 중복 | 분반 중복 | 표시 | 표시 |
| 학년 중복 | 학년 중복 | 표시 | 표시 |
| 같은 과목 같은 요일 | 같은 요일 중복 | 표시 | 표시 |
| 교수 불가능 시간 | 교수 불가시간 | 표시 | 표시 |
| 고정 시간 위반 | 고정시간 이탈 | 표시 | 표시 |
| 교수 강의실 일관성 | 강의실 불일치 | 표시 | 표시 |
| 시간대 위반 | 시간대 위반 | 표시 | 표시 |
| 보관함 미배치 블록 | 해당 없음 | 별도 표시 없음 | 표시 |

## 표시하지 않는 항목

아래 항목은 수동편집 화면의 위반 표시 대상에서 제외한다.

| 위반 항목 | 시간표 작은 글자 | 오른쪽 위반사항 | 전체 검증 | 비고 |
|---|---|---|---|---|
| 점심시간 배치 | 표시 안 함 | 표시 안 함 | 표시 안 함 | 점심시간도 수동편집에서 빈 칸처럼 사용할 수 있게 둔다. |
| 고정 강의실 위반 | 표시 안 함 | 표시 안 함 | 표시 안 함 | 수동편집에서는 강의실 변경 자유도를 우선한다. |
| 과목 불가 강의실 | 표시 안 함 | 표시 안 함 | 표시 안 함 | 수동편집 표시에서는 제외한다. |
| 블록 시작 교시 위반 | 표시 안 함 | 표시 안 함 | 표시 안 함 | 수동편집에서는 시작 위치 자유도를 우선한다. |
| 교수 불가 강의실 | 삭제됨 | 삭제됨 | 삭제됨 | 해당 기능은 더 이상 사용하지 않는다. |

## 개선 후보

1. 시간표 작은 글자와 전체 검증의 명칭을 완전히 동일하게 맞추면 사용자가 더 빨리 이해할 수 있다.
2. 오른쪽 위반사항에서 각 항목을 클릭하면 시간표의 해당 블록이 더 강하게 강조되면 좋다.
3. 전체 검증의 정상 항목은 접고, 비정상 항목만 먼저 펼치면 확인 시간이 줄어든다.
4. `시간대 위반`은 사용자가 의미를 바로 알기 어렵기 때문에 `학부/대학원 시간대 위반`처럼 더 구체적인 이름으로 바꾸는 것이 좋다.
